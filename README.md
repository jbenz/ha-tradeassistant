# TradeAssistant Home Assistant Add-on Repository

Professional cryptocurrency trading and market monitoring add-on for Home Assistant.

## Installation

1. Go to **Settings** → **Add-ons** → **Add-on Store**
2. Click the **⋮** (menu) button in the top right
3. Select **Repositories**
4. Add: `https://github.com/madrot/ha-tradeassistant`
5. Close the dialog
6. Search for **TradeAssistant** in the add-on store
7. Click **Install**

## Add-ons in this repository

### TradeAssistant
- Real-time market price feeds (WebSocket + REST API)
- Professional Lovelace dashboard card
- Portfolio tracking (optional)
- Trade execution support (optional)
- SQLite persistence
- Multi-architecture support (amd64, arm64, armv7, i386)
